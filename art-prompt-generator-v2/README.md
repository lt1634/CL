# DSE 視藝 卷一 Part B 題目生成器 v2

專為**香港 DSE 視藝科**考生準備**卷一 Part B**（有限制題目）而設的題目生成器。唔係 AI 生圖用嘅 prompt 工具，而係模擬考題 + 對齊四項評分準則嘅練習輔助。

---

## 功能

- **命題**：隨機模擬 DSE 風格有限制題目（如「成長」「衝突」「循環」「距離」「界限」等）
- **六塊輸出**：命題、Concept 諗法、視覺元素建議、構圖提示、技法建議、扣題引導
- **常見犯錯／注意**：部分題目附「犯錯／注意」提示，強調乙部須聯繫甲部評賞作品的物理特性或意涵，並運用構圖與光源策略提升作品質量
- **評分對照**：每塊標示對應 DSE 評分範疇（媒材及技法、視覺元素及設計原理、創意與想像力、主題傳意，各 20%）
- **簡潔模式**：只顯示命題 + 合併方向提示，方便快速睇題
- **收藏 / 複製 / 分享**：儲存心水題目、複製全文、分享
- **最近生成**：歷史記錄，可點擊還原
- **中英切換**：題目庫支援中文、英文
- **香港時間**：頁面顯示香港時區
- **PWA**：可安裝、離線使用（manifest + service worker）

---

## 使用方法

1. 用瀏覽器打開 `index.html`，或部署到靜態空間（如 GitHub Pages、Vercel）
2. 按 **「✨ 生成題目」** 隨機取得一條 DSE 風格題目
3. 可切換 **簡潔模式** 只睇命題與方向提示
4. 需要時按 **★ 收藏**、**📋 複製** 或 **↗ 分享**

---

## 評分準則對照（四範疇各 20%）

| 評分範疇 | 對應輸出區塊 |
|----------|----------------|
| 媒材及技法的選擇和運用 | 技法建議 |
| 視覺元素及設計原理的選擇和運用 | 視覺元素建議、構圖提示 |
| 創意與想像力 | 命題、Concept 諗法 |
| 主題傳意 | 扣題引導 |

## 題目來源與邏輯

- **歷年 DSE 試題** (2013–2025)：命題風格參考卷一 Part B 真題（如時空限制、具體與抽象結合、感官轉譯、視點與心理）。
- **《摘星天書》**：部分題目附 **★ 星級提示 (starTips)**，對應策略 1 構圖透視、策略 2 光源設計。
- **甲部聯繫（評賞作品風格引導）**：部分題目附 **partALink**，融入 2014–2022 歷屆卷一甲部評賞作品風格（媒材與技法、構圖透視、光源設計、符號與意義），供乙部創作對應；例如畢加索、拉圖爾、霍克尼、金昌烈、馬格利特等。
- **色彩調性 (colourTone)**：部分題目附預設色彩建議（如冷色調、鄰近色、高飽和度），對應天書「色彩構成」。
- **常見犯錯／注意 (commonMistakes)**：部分題目附「犯錯」與「注意」列表，提醒考生避免常見失誤（如內外分界模糊、忽略甲部聯繫、光源過於平均等），並對應構圖與光源策略。
- 構圖提示區塊標示「參考：策略1 構圖透視」；星級提示區塊標示「參考：策略2 光源設計等」。

## 檔案結構

- `index.html` — 主頁與邏輯
- `prompts-dse.js` — 題目庫（約 26 條，中英各一，每條含六塊；部分含 starTips）
- `manifest.webmanifest` — PWA 設定
- `service-worker.js` — 離線快取
- `icon.svg` — 應用圖示

## 與 v1 嘅分別

- **v1**：通用藝術靈感，一題目 + 一 hint，五類（抽象、自然、情感、人物、技巧）
- **v2**：DSE 卷一 Part B 專用，命題 + 六塊（對齊評分），模擬有限制題目，目標用戶係考生與老師
