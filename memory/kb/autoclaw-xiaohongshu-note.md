## 大意

- **AutoClaw**：基於 OpenClaw + 智谱，標榜「一鍵部署、打開即用」。
- 從下載到在 **飞书（Feishu）** 跑通第一個 Agent，描述為「幾分鐘搞掂」。
- 針對 OpenClaw 常見門檻（終端、開發者帳號、API Key），用 **GUI + RPA + 內建模型** 降低門檻。

## 核心體驗（原文）

1. **唔使掂終端**：點擊 GUI 就搞掂。
2. **一鍵接飞书 Bot**：幾秒接好，手機／電腦多端同步。
3. **Skill 內建**：點一下就裝好。
4. **活動詳情面板**：每步執行過程可視化，出錯易定位。
5. **隨時換模型**。

## 場景

- AutoClaw 主要「活在飞书裡」，到處都可指揮佢做嘢。
- 屋企有閒置電腦就裝住長開，當 24 小時私人助手。

## 同你而家 setup 嘅關係

- 你而家：用 **global OpenClaw**（終端、openclaw.json、WhatsApp 等），有門檻但可控。
- AutoClaw：面向「唔想掂終端」嘅用戶，用飞书 + GUI + 智谱內建。
- 若你之後想試 **飞书** 或 **少碰終端**，可以當參考；若只係想知有呢個產品，留呢份筆記即可。
