## 分層與保留

| 層級 | 內容類型 | 保留天數 | 整理方式 |
|------|----------|----------|----------|
| **P0** | 身份、核心偏好、永遠不要再 | 永久 | 只增不刪，定期精煉 |
| **P1** | 血淚教訓、重要決定、專案規則 | 長期 | 每週檢視，過時移 archive |
| **P2** | 近期糾正、一次性偏好、跟進備忘 | 7–30 日 | 每週 compact 到 memory/archive/ 或由 weekly cron 歸檔 |

- **每週 compact**：把 P2 中已過保留期或已解決嘅項，搬入 `memory/archive/YYYY-MM-DD.md`，避免 MEMORY.md 無限變長。
- 可設 weekly cron：「讀 MEMORY.md，把過期 P2 寫入 memory/archive/，再從 MEMORY.md 刪除該段。」
