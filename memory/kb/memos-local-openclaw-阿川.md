## 問題

OpenClaw（龙虾）**唔會跨 session 記住嘢**：新窗口問「按我之前的格式来生成」，佢會問「之前的格式是？」。要重複教、重複踩坑。

---

## 解法：MemOS 本地記憶

- **一句安裝**：`npm install -g @memtensor/memos-local-openclaw`，然後 `openclaw gateway restart`。
- **記憶面板**：瀏覽器開 http://127.0.0.1:18799 ，設密碼，完成。對話會自動記入去。
- **100% 本地**：SQLite 檔喺本機，零上傳；備份 = 複製檔、遷移 = 拷檔、刪除 = 刪檔。適合離線或要數據主權嘅場景。
- **效果（官稱）**：Token 省約 49%、任務速度約 2 倍、真係會「長腦」— 新 session 都記得之前講過咩。

---

## 四個任務（阿川示範）

1. **裝備記憶**：裝好 + 開 18799 設密碼，聊幾句再 refresh 面板，見到記錄即 OK。
2. **記憶復活**：先同龙虾講「下個月喺深圳搞 OpenClaw 沙龙 4月5号」；開 **new** 新 session 問「我下个月要在哪里举办什么活动？」— 佢會答返深圳、OpenClaw 沙龙、4月5号。關鍵資訊會抽出來存記憶庫，新窗口都可檢索。
3. **技能自動生成**：例如同龙虾搞 30 分鐘先搞掂「每日簡報」格式；MemOS 會把踩坑過程**提煉成技能**。之後再問，直接調用該技能，唔使再教。若你後來有更好寫法，MemOS 會**自動升級**該技能。
4. **公會技能（多龙虾共享）**：多隻 agent（例如 Milo / Josh / Angela / Bob）可**共用同一個記憶池**。Bob 沉澱嘅「代碼審查 checklist」會變成共享技能；Angela 之後要檢查時，佢嘅龙虾會自動用到 Bob 嘅思路，唔使再摸 2 小時。即：**個人記憶**（草稿、臨時思路）只係自己；**共享記憶池**（技能、標準做法）全部龙虾都用到。

## 數據與隱私

- 一個 **SQLite 檔**喺你電腦。想備份就複製；想遷移就拷過去；想刪就刪檔。
- 完全自主，零外部依賴。離線環境（例如本地 Ollama + 唔連外網）都可照用記憶積累知識庫。

## 快速試

1. `npm install -g @memtensor/memos-local-openclaw`
2. `openclaw gateway restart`
3. 開 http://127.0.0.1:18799 完成配置（設密碼）
4. 同龙虾聊幾句，過幾日再問「还记得吗」— 若記得，即係記憶生效。

---

## 同你而家 memory 嘅關係

你而家已有 **MEMORY.md、memory/YYYY-MM-DD、AGENTS.md** 等，係「手動／規則型」記憶。MemOS 係**多一層**：自動抽關鍵資訊、自動生成技能、可多 agent 共享、本地 SQLite。若你覺得「龙虾總係失憶」或想多 agent 共享 checklist／格式，可以試加 MemOS；若你主要靠 MEMORY.md + 每日 log 已夠，可以當備選方案記住就得。
