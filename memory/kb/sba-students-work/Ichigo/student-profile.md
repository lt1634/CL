# Ichigo（中五 SBA）

> 由 `sba-students.md` 索引連結至此；**更新進度請改本檔**。分拆前全文見 `memory/archive/sba-students-2026-04-03-monolithic.md`。

- **年級**：高中（SBA 進行中）
- **總體主題**：**自我投射與角色演變 (Self-Projection & Fursona)** —— 透過非人類角色（兔、狼等獸類混合）表達人類情感。
- **核心構思**
  - 角色「Kun」代表對「孤獨」同「被愛」嘅渴望（Solitude, seeking for comfort）。
  - 探討「面具」背後的真實身份：穿上獸裝係逃避，定係尋找真實自我？
- **作品形式**：結合 2D 角色設計圖、分鏡圖，以及 3D 實體毛皮頭部（Fursuit Head）製作。
- **進度（Second Development）**
  - **已完成**：角色重新設計（Vexar Xaeros、新版 Kun 彩色草圖）；立體計劃（Fursuit 材料 Foam core / hot glue / fur、Milestone）；情感描述（角色基於孤獨與尋求安慰）。
  - **待加強**：Critical Study（連結「變形」「面具」「身份認同」相關藝術家）；實驗記錄（製作 3D 頭部時要影低每階段如骨架期、貼皮期入 Workbook）。
  - **📸 Dropbox 最新（2026-03-26）**：今日上傳 IMG_8428-8427.jpeg（SBA2 新相片）+ ClassNotes-2026-03-26.md
- **特性**
  - 次文化專才：極鍾意 Furry（獸迷）文化，對角色設計有強烈偏執。
  - 低動機但高專注：對唔感興趣嘅嘢唔想郁，對自己角色（如 Kun）可以反覆修改。
  - 立體潛力：計劃做 Fursuit Head，展現超越一般學生嘅立體空間感同工藝耐性。
  - 情感投射：角色代表「孤獨、尋求愛、內心掙扎」。
- **督導策略**
  - **尊重其專業**：傾獸迷文化，讚角色設定好 Deep，佢會當你「圍內人」更肯聽。
  - **以實幹代筆墨**：唔鍾意寫字就叫佢影低製作過程，旁邊加 3–5 個單字標籤（Foam, Structure, Texture）。
  - **引入專業參考**：電影特效（如 *Star Wars* 外星人造型）或當代動物擬人化藝術家，將興趣同專業藝術掛鉤。
- **本日目標建議（可作下一堂任務）**
  - 任務 1：**「表情符號」練習** —— 喺 Workbook 畫角色「Kun」嘅 3 種情緒（喜、怒、哀），方便稍後做立體頭部時決定神態。（約 20 分鐘）
  - 任務 2：**材料實驗 (Texture Test)** —— 攞 2–3 種唔同質地布料或紙皮，試熱熔膠點貼得最穩，將實驗品貼入 Workbook。（約 30 分鐘）
- **給老師的建議（如何開始第一堂）**
  - **肯定小眾**：「你個 Fursuit 如果整得成，會係全級最勁嘅 SBA 立體作品。」對佢好有動力。
  - **管理難度**：計劃宏大但能力可能有限，提醒「由細做起」(Start Small)——先做一對耳朵或簡單面具骨架，確保每堂都有嘢交。
  - **連結 2028 SBA 準則**：HKEAA 好睇重 Media Experiment；叫佢一定要影低「剪 Foam」「修剪毛皮」等過程相，就係 Workbook 核心內容。
  - **下次上堂**：同助手講「下堂 Ichigo，幫我推佢做角色表情練習同材料實驗，提醒佢影低 Fursuit 製作過程。」助手就會幫你搞掂。
