# SBA 助手 Workspace 範本（Berryxia / Saboo 三層架構）

將呢個 folder 入面嘅 **IDENTITY.md、USER.md、AGENTS.md、MEMORY.md** 複製去你 **SBA agent 的 workspace 根目錄**（例如 `~/.openclaw/workspace-sba/`），與 **SOUL.md** 同層。

- **SOUL.md** — 你要自己放（完整 SBA system prompt）。
- 複製後請編輯 **USER.md**，填老師名、時區等。
- **MEMORY.md** 留空即可，之後同助手講嘢、畀 feedback 時佢會按 AGENTS.md 規則寫入。

同一套三層架構亦適用於 **1號 agent**（主 agent），範本在 **main-agent-workspace-templates/**。詳情見 **sba-agent-berryxia-guide.md**。
