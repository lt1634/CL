## 一句核心

一個玩 AI 兩個月嘅高中生，做出嘅結果可能超過玩兩年嘅人；作者見完之後反思：**學歷仲有冇用？高中仲有冇必要上？** 結論偏向：時代把機會塞到手裡，**行動力、被看到、商業認知、不安分** 比「先學完再做」重要。

---

## 嗰個高中生做咗咩

1. 參加某大廠黑客松拿獎，入大廠實習，每週去一兩次。  
2. 小紅書幾條片，累積近一萬粉（作者當場睇，唔係造假）。  
3. 每日七八個鐘用 **OpenClaw**，一半學習、一半遠程操控電腦做任務；用 AI 破解學校學習平板，喺學校經 OpenClaw 入 Telegram 工作。  
4. 獨立開發產品，得到學校、大廠支持。  
5. 完全冇手寫過 code，接觸 AI 寫 code 都係兩個月。  

作者形容：對商業、產品嘅認知已經好成熟；做自媒體有「AI 專家」嘅演出感，感染力強。

## 點解係佢？四點

| 點 | 內容 |
|----|------|
| **行動力** | 冇「先學完再做」——黑客松、產品、自媒體一齊做。好多人諗三年唔敢開始嘅嘢，佢兩個月就做。 |
| **理解「被看到」** | 最珍貴唔係內容本身，而係能被看到。注意力有限，酒香也怕巷子深 → 所以認真做自媒體，幾條片萬粉。 |
| **商業認知超齡** | 會提孫宇晨、為達目標不計代價；玩 AI + 自媒體 + web3，喺活動裡鏈接人脈；全方位布局，好多成年人都冇。 |
| **保持「不安分」** | 用 AI 破解學校平板、喺學校用 OpenClaw 工作——呢種不安分，作者話係 AI 時代最稀缺。 |

## 呢個時代俾咩機會（作者歸納）

1. **技術唔再係唯一壁壘**：唔識寫 code 都能開發產品、拎黑客松獎。AI 把「會用工具」嘅價值放大；知識平權化，個人稀缺嘅係想像力、創造力、品味、同把知識應用到現實問題嘅能力。  
2. **速度決定生死**：想法到 MVP 可以幾日搞掂；Vibe Coding 令一個人做齊調研、設計、開發、營銷。成本低、回報上限高。兩個月做到嘅嘢，五年前可能要團隊做一年。  
3. **年輕人時間窗口**：20+ 學習成本低、時間多、冇路徑依賴。企業用人分化成兩類——基本功極紮實（資深開發/算法）vs **AI 工具用得順**（涉獵廣、上手快、能拆解任務、組合工具鏈、先做出 70 分再迭代）。後者「1 天 0→70 分」嘅價值在 AI 時代被放大。  
4. **「被看到」係核心能力**：做出來只係第一步，讓人看到先係關鍵。代碼是冷的，故事是熱的；講好代碼背後嘅故事係獨立開發者必修。  

---

## 作者俾年輕人嘅建議

1. **越早開始越好**：條路仲未擠；有容錯、有兜底、有時間試錯。  
2. **唔好等「準備好」**：邊做邊學，實戰中成長；好多人等三年仲喺度糾結。  
3. **全方位布局**：唔止技術，仲有自媒體、人脈、商業認知；超級個體會係一個人完成調研、設計、美術、開發、營銷。  
4. **保持「不安分」**：呢種勁頭先係 AI 時代最稀缺。  

文末：**這個時代真的在把機會往我們手裡塞。現在就開始，不要等，不要怕。幹就完了。**

## 同 CL / 教學嘅對照

- 你教 AI 課程、用 OpenClaw 做 SBA 助手同個人自動化：可以同學生講「玩 AI 兩個月嘅高中生」嘅例子——**行動 > 等準備好**、**被看到**（內容 + 曝光）、用 OpenClaw 做真實任務而不只 demo。  
- 「唔識寫 code 都能開發、拿獎」同左子禎那篇「指令 vs 目標」一致：門檻係意圖同組合工具，唔係手寫代碼。  
- 若學生有「不安分」、肯用工具（包括在校內合理範圍用 OpenClaw），可以引導成「速度 + 被看到 + 商業認知」，而唔係只談學歷有冇用。
