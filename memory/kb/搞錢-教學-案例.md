# 搞錢 — 提示詞 / 企業 workshop 案例庫

**係咩**：企業內訓、半日實戰工作坊——audit 團隊點用 AI、優化 prompt、即時提升產出。賣嘅係「賣鏟」：公司有 AI 訂閱但唔識用，你教佢哋點溝通、點 chain 工具。

**賣俾邊個**：已買 ChatGPT/Claude/其他年費但員工輸出好水嘅公司；想「全公司會用 AI」嘅老闆。

**收幾多**：半日 workshop 幾千至萬幾；audit + 優化方案可再報。

**案例**：（每週 agent 可從 session 抽「改進前後」嘅 prompt 例子、常見錯誤，你審核後貼入；對外可脫敏做銷售素材。）
